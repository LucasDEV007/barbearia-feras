# Barbearia Feras — Sistema de Agendamento

Sistema web para agendamento de horários em barbearias.

## Funcionalidades

- Agendamento de horários
- Seleção de serviços
- Painel administrativo
- Cancelamento de agendamentos

## Tecnologias

- React
- Tailwind CSS
- Supabase
- Lovable
